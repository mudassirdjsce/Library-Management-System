
# DJSCE Library Management System

A modern, responsive web application for managing library operations at DJSCE (Dwarkadas J. Sanghvi College of Engineering).

## Description

The DJSCE Library Management System is a comprehensive solution designed to enhance the library experience for students and faculty. Built with React and Vite, this application provides a user-friendly interface for browsing, searching, borrowing, and purchasing books. The system features a dynamic catalog with real-time search capabilities, user authentication, and a responsive design that works across all devices.

## Features

- **User Authentication**: Secure login and registration system for library members
- **Book Catalog**: Extensive collection of books with detailed information and cover images
- **Advanced Search**: Smart search with autocomplete functionality and genre filtering
- **Book Management**: Borrow, return, and reserve books with ease
- **Purchase Books**: Integrated payment system for purchasing books directly
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Dynamic Backgrounds**: Engaging background slideshow on the catalog page
- **User Dashboard**: Personalized dashboard to track borrowed and reserved books
